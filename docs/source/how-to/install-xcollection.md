# Install xcollection

xcollection can be installed in three ways:

```{eval-rst}
.. tab:: pip

    Using the `pip <https://pypi.org/project/pip/>`__ package manager:

    .. code:: bash

        $ python -m pip install xcollection

.. tab:: conda

    Using the `conda <https://conda.io/>`__ package manager that comes with the
    Anaconda/Miniconda distribution:

    .. code:: bash

        $ conda install xcollection --channel conda-forge

.. tab:: Development version

    To install a development version from source:

    .. code:: bash

        $ git clone https://github.com/NCAR/xcollection
        $ cd xcollection
        $ python -m pip install -e .

```
